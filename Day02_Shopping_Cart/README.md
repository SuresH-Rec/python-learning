# 🛒 Shopping Cart System

## 📖 Project Overview

This is my second Python mini project.

The application stores multiple products, calculates the total amount for each product, categorizes the purchase based on the total price, and displays a shopping summary.

---

## ✨ Features

- Stores multiple products using dictionaries
- Calculates total price (Price × Quantity)
- Categorizes purchases:
  - Premium
  - Expensive
  - Budget
  - Economy
- Displays a detailed shopping report
- Uses functions to improve code reusability

---

## 🛠️ Technologies Used

- Python 3
- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements
